# 批处理脚本使用说明

## 📋 batch_download_and_backfill.py

### 功能

自动执行数据下载和 PE 回填的完整流程。

### 执行步骤

**步骤 1**: 重置并下载所有历史数据（自动执行）
- 脚本: `reset_and_redownload_all.py`
- 功能: 下载所有股票的历史数据
- ETL: 自动计算 PE/EPS

**步骤 2**: 添加新资产（自动执行）
- 脚本: `add_new_asset_complete.py`
- 功能: 将新资产添加到数据库

**步骤 3**: 回填历史 PE（需要确认）⚠️
- 脚本: `backfill_valuation_history.py`
- 功能: 重新计算所有历史 PE 数据
- 确认: 需要用户手动确认是否执行

### 使用方法

```bash
# 运行批处理
python3 batch_download_and_backfill.py
```

### 何时运行步骤 3

**需要运行**:
- ✅ 首次设置系统
- ✅ 财报数据更新后需要重新计算历史 PE
- ✅ 发现历史 PE 数据不准确

**不需要运行**:
- ❌ 步骤 1 已成功（ETL 已自动计算 PE）
- ❌ 只是日常数据更新

### 注意事项

1. **执行时间**: 完整流程可能需要 30-60 分钟
2. **网络要求**: 需要稳定的网络连接
3. **中断恢复**: 可以随时 Ctrl+C 中断
4. **数据备份**: 建议运行前备份数据库

### 快捷方式

如果只需要运行单个脚本：

```bash
# 只下载数据
python3 reset_and_redownload_all.py

# 只添加新资产
python3 add_new_asset_complete.py

# 只回填 PE
python3 backfill_valuation_history.py
```
