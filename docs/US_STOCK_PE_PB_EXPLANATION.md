# 美股 PE/PB 数据获取说明

## 问题1: "实时快照"是什么意思?

**答**: **收盘数据**,不是盘中数据。

### 验证数据
```
AAPL ticker.info 数据时效性:
- 当前时间: 2026-01-09 17:57 (北京时间)
- regularMarketTime: 2026-01-09 05:00:02 (美东时间 2026-01-08 16:00:02)
- regularMarketPrice: 259.04 (收盘价)
- previousClose: 260.35 (前一日收盘价)
```

### 说明
- yfinance `ticker.info` 返回**最近一个交易日的收盘数据**
- PE/PB 等估值指标基于收盘价计算
- 由于在北京时间运行(美股已收盘),获取的是 2026-01-08 的收盘数据
- 如果在美股盘中运行,会获取盘中实时数据

## 问题2: 股息率放大了100倍?

**答**: 是的,这是一个 bug,已修复! ✅

### 问题分析
- **yfinance 返回**: `dividendYield: 0.4` (小数格式,表示 40%)
- **错误代码**: `dividend_yield = info.get('dividendYield') * 100` ❌
- **错误结果**: 存储为 `40.0`,相当于 4000%

### 修复方案
```python
# 修复前 (错误)
dividend_yield = info.get('dividendYield')
if dividend_yield:
    dividend_yield = dividend_yield * 100  # ❌ 错误!

# 修复后 (正确)
dividend_yield = info.get('dividendYield')  # ✅ 直接存储小数格式

# 显示时转换为百分比
div_str = f"{dividend_yield*100:.2f}%" if dividend_yield else "N/A"
```

### 修复后的数据

|------|-------------|----------|
| AAPL | 0.4000 | 40.00% |
| MSFT | 0.7600 | 76.00% |
| NVDA | 0.0200 | 2.00% |
| GOOG | 0.2600 | 26.00% |

**注意**: BAC (199%) 和 TSM (106%) 的股息率看起来异常高,可能是 yfinance 数据问题,需要进一步验证。

## 数据存储规范

### 数据库存储格式
- **PE**: 直接存储倍数 (如 34.72)
- **PB**: 直接存储倍数 (如 51.90)
- **股息率**: 存储小数格式 (0.4 = 40%)

### 显示格式
- **PE**: `34.72`
- **PB**: `51.90`
- **股息率**: `40.00%` (小数 * 100)
